## Core conceit (working)

| Element | Intent |
| --- | --- |
| **Rune Path** | Not a single road — a **network** of marks, people, and trial-and-error corridors out of Tarro's reach. |
| **Carved spells** | Runes **cut or burned** into trees, stone, bridge pilings, barn beams — latent until another Magicborn **speaks or feeds** them; teach techniques, warnings, or one-shot escapes. |
| **Pedagogy** | The Path **trains** as it moves you: each leg expects slightly more skill than the last, or punishes hubris. |
| **Risk** | Marks can be **found by the House**; helpers can be **turned**; wrong syllables **burn** the learner. |
