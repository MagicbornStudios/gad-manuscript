| `MAGICBORN-DOCS-SHORT-STORY-LORE` | `accepted` | Magicborn docs should read like short story fragments, lore scenes, or compact canon notes instead of dry encyclopedia entries | the section can stay readable to visitors while still serving as worldbuilding input for chat |
| `MAGICBORN-DOCS-WORLDBUILDING-NOT-MANUSCRIPT` | `accepted` | the section is for world, character, faction, and setting material, not raw novel chapters | lore stays distinct from manuscript drafts and avoids accidental ingestion of reader-facing chapter text |
